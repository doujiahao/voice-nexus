# 智能客服话务平台 — 接口契约文档

> 本文档定义统一服务端对外暴露的全部接口，供 Web 前端、freeswichService 团队对接使用。
>
> - Web 前端使用 `Authorization: Bearer <access_token>` 认证
> - freeswichService 使用 `X-Internal-Key: <key>` 认证
> - 基础路径：`/api/v1`（外部）、`/internal`（内部）

---

## 一、内部接口（freeswichService → 服务端）

### 1.1 呼入路由申请

```
POST /internal/calls/{fs_call_id}/route
Header: X-Internal-Key
```

**请求**：
```json
{
  "customer_phone": "13800138000",
  "called_number": "95598",
  "skill_group": "default",
  "fs_metadata": {
    "caller_id_name": "13800138000",
    "destination_number": "95598",
    "network_addr": "192.168.1.100"
  }
}
```

**响应 — 分配成功**：
```json
{
  "success": true,
  "call_session_id": "uuid-session-001",
  "route_action": "RING",
  "target_agent_id": "uuid-agent-001",
  "target_extension": "8001",
  "ring_timeout_sec": 30,
  "customer": {
    "id": "uuid-customer-001",
    "name": "张三",
    "account_no": "1234567890",
    "meter_no": "M001234"
  }
}
```

**响应 — 排队**：
```json
{
  "success": true,
  "call_session_id": "uuid-session-001",
  "route_action": "QUEUE",
  "queue_position": 3,
  "estimated_wait_sec": 45
}
```

**响应 — 失败**：
```json
{
  "success": false,
  "error_code": "NO_AVAILABLE_AGENT",
  "message": "技能组无可用坐席且队列已满"
}
```

### 1.3 音频帧上行（WebSocket）

```
WS /internal/ws/audio
Header: X-Internal-Key (在连接握手时传递)
```

**上行数据**：Protobuf 二进制（AudioFrame）

```protobuf
message AudioFrame {
  bytes  data            = 1;  // 完整 WAV 文件字节（单声道，一句话）
  string speaker_id      = 2;  // 说话人 UUID
  string speaker_name    = 3;  // 说话人显示名
  string speaker_role    = 4;  // "customer" | "agent"
  string language        = 6;  // "zh"
  bool   is_last         = 7;  // true 表示本轮最后一包
  uint32 sequence_number = 8;  // 单调递增序号
}
```

**下行 ACK**（JSON）：
```json
{
  "sequence": 42,
  "accepted": true,
  "call_session_id": "uuid-session-001",
  "turn_id": "uuid-turn-042"
}
```

> 注意：连接建立时需通过 query param 传递 `fs_call_id`，如 `/internal/ws/audio?fs_call_id=xxx`

## 二、外部 REST 接口（Web 前端 → 服务端）

> 以下为查询/管理类操作，走 HTTP REST。

## 三、WebSocket 接口（服务端 ↔ Web 前端）

### 3.1 连接与认证

```
WS /ws/call-events
连接时 Query: /ws/call-events?token=<access_token>
```

### 3.2 心跳

- 客户端每 20s 发送 `{"type": "ping", "ts": 1715400000000}`
- 服务端回复 `{"type": "pong", "ts": 1715400000000}`
- 60s 无任何消息视为断线，客户端自动重连

### 3.3 Client → Server（坐席实时操作，带 request_id 应答）

#### 接听来电
```json
{
  "type": "call_accept",
  "call_session_id": "uuid-session-001",
  "request_id": "req_001"
}
```

**应答**：
```json
{
  "type": "call_accept_ack",
  "request_id": "req_001",
  "success": true,
  "call_session_id": "uuid-session-001",
  "status": "ANSWERED",
  "accepted_at": "2026-05-11T09:15:30+08:00"
}
```

**失败应答**：
```json
{
  "type": "call_accept_ack",
  "request_id": "req_001",
  "success": false,
  "error_code": "SESSION_ALREADY_ENDED",
  "message": "通话已结束"
}
```

#### 拒接来电
```json
{
  "type": "call_reject",
  "call_session_id": "uuid-session-001",
  "request_id": "req_002"
}
```

**应答**：
```json
{
  "type": "call_reject_ack",
  "request_id": "req_002",
  "success": true
}
```

#### 挂断通话
```json
{
  "type": "call_hangup",
  "call_session_id": "uuid-session-001",
  "reason": "NORMAL",
  "request_id": "req_003"
}
```

**应答**：
```json
{
  "type": "call_hangup_ack",
  "request_id": "req_003",
  "success": true,
  "call_session_id": "uuid-session-001",
  "status": "ENDING"
}
```

#### 坐席状态切换
```json
{
  "type": "agent_status_change",
  "status": "ONLINE",
  "request_id": "req_004"
}
```

可选值：`ONLINE` / `REST` / `OFFLINE`

**应答**：
```json
{
  "type": "agent_status_change_ack",
  "request_id": "req_004",
  "success": true,
  "status": "ONLINE",
  "status_since": "2026-05-11T09:00:00+08:00"
}
```

**失败应答**（如通话中不能切离线）：
```json
{
  "type": "agent_status_change_ack",
  "request_id": "req_004",
  "success": false,
  "error_code": "AGENT_BUSY",
  "message": "通话中不能切换为离线"
}
```

## 四、错误响应统一格式

所有 REST 接口错误响应遵循：

```json
{
  "success": false,
  "error_code": "AGENT_NOT_FOUND",
  "message": "坐席不存在",
  "timestamp": "2026-05-11T09:12:00+08:00"
}
```

**通用错误码**：

| 错误码 | HTTP Status | 说明 |
|---|---|---|
| UNAUTHORIZED | 401 | Token 无效或过期 |
| FORBIDDEN | 403 | 无权限 |
| NOT_FOUND | 404 | 资源不存在 |
| INVALID_PARAM | 400 | 参数校验失败 |
| INTERNAL_ERROR | 500 | 服务端内部错误 |
| AGENT_NOT_FOUND | 404 | 坐席不存在 |
| AGENT_BUSY | 409 | 坐席忙碌中 |
| SESSION_NOT_FOUND | 404 | 通话会话不存在 |
| SESSION_ALREADY_ENDED | 409 | 通话已结束 |
| NO_AVAILABLE_AGENT | 503 | 无可用坐席 |
| QUEUE_FULL | 503 | 排队队列已满 |
| INVALID_ACTION | 400 | 无效操作（如对已结束通话执行接听） |
| INTERNAL_KEY_INVALID | 401 | 内部服务认证失败 |
